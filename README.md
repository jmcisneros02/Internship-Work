# Institutional Research and Strategic Analytics (IRSA) Internship Project
This project investigates where students go after leaving CSU Chico by analyzing enrollment data from the National Student Clearinghouse (NSC). The central research question: If students leave Chico State, where are they transferring?

## Project Contents
### Shiny App (app.R) 
An interactive R Shiny application that visualizes:
* The top 10 institutions students transferred to, filterable by academic year
* Students' last enrollment term at CSU Chico for each academic year

https://jmcisneros.shinyapps.io/fake-nsc-data-shiny-app/ 

### Simulated Data (fake_data.qmd)
This file contains the process of taking the raw data and generating a simulated dataset for demonstration purposes.

* # Note on Data Privacy:
All data displayed in this repository is simulated. Due to FERPA regulations, the original exploratory analysis and raw data are confidential and not included in this repository.
